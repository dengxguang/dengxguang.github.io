## 如何让页面变得丝滑

### 什么是 Web 性能

Web 性能是指网站内容在网页浏览器中的加载和渲染速度，以及对用户交互的响应。

### 页面丝滑有什么衡量标准

#### FPS

#### RAIL

| RAIL |  用户操作 | 关键指标 |
| ------- | ---------- | -----------|
| 响应（Response） | 点击按钮。                    | 小于100ms|
| 动画（Animation）| 滚动页面，拖动手指，播放动画等。  | 小于16ms |
| 空闲（Idle）     | 用户没有与页面交互，但应该保证主线程足够处理下一个用户输入。| 小50ms |
| 加载（Load）     | 用户加载页面并看到内容。 | 1000ms|
